# Alex Styles Portfolio Project

A high-fidelity, responsive, and interactive portfolio website designed for a UI/UX designer. This project demonstrates the application of modern design principles, accessibility best practices, and the integration of AI-assisted development using Microsoft Copilot.

## Features

- **Foundational CSS**: A robust design system using CSS variables and modern layout techniques (Flexbox & Grid).
- **Responsive Design**: Mobile-first approach with fluid typography and adaptive layouts for all screen sizes.
- **Interactive UI/UX**: Hover effects, accessible modals for project details, tooltips, and smooth scroll interactions.
- **Visual Enhancements**: Glassmorphism effects, gradient text, and subtle micro-animations for a premium feel.
- **Accessibility**: Semantic HTML5 markup, ARIA roles, and focus management for keyboard navigation.

## Technologies Used

- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)
- Google Fonts (Inter & Outfit)

## How to View

1. Clone this repository.
2. Open `index.html` in any modern web browser.
3. To view the mobile layout, use the browser's developer tools and toggle device emulation.

## AI Assistance (Microsoft Copilot)

Microsoft Copilot was utilized as a highly capable pair programmer throughout the development of this portfolio. Its assistance was instrumental in several key areas:

- **Foundational CSS & Design System**: Copilot assisted in defining a consistent CSS variable architecture for colors, spacing, and typography. It suggested the 'Inter' and 'Outfit' font pairing and provided the initial scaffold for the glassmorphism effects used in the header and cards.
- **Responsive Design & Media Queries**: Copilot was instrumental in generating mobile-first media queries. It suggested optimal breakpoints and helped implement fluid layout transitions using CSS Grid, ensuring the portfolio remains fully functional on devices ranging from smartphones to large desktop monitors.
- **Interactive UI/UX Features**: Interactive elements such as the project detail modals and tooltips were developed with Copilot's guidance. It provided the JavaScript scaffold for modal focus management and suggested the implementation of CSS-based tooltips to minimize JavaScript overhead.
- **Visual Enhancements**: To achieve a polished and professional design, Copilot suggested the use of subtle gradients, backdrop filters for glassmorphism, and micro-animations. These enhancements create a premium feel that aligns with modern UI/UX standards.
- **Accessibility Integration**: Copilot assisted in implementing accessibility best practices, including semantic HTML5 markup and ARIA attributes, ensuring the site is navigable for all users.
